重新安装 Ubuntu 24.02 LTS 系统

Posted on 2024-08-19

由于旧系统崩溃无法开机进入，尝试多种方法无果，因此在双系统下重新安装 Ubuntu 系统

硬件：[微星 B550M]-[AMD R9-5900X]-[Ubuntu 22.04]-[Windows 10]

卸载 Ubuntu

  1. 备份 Ubuntu 系统文件
  2. 开机时连续按 Del 键，进入 BIOS 系统，进入 Setting > Boot 修改开机启动项
  3. Windows系统下，删除 win10 EFI 中的 Ubuntu 文件
    1. Win + R 运行 diskpart
    2. list disk 查看磁盘，select disk 0 选择磁盘 0
    3. list partition 查看分区，select partition 1 选择 EFI 系统分区
    4. assign letter=p 分配盘符
    5. 由于该盘符无访问权限，通过管理员身份运行 记事本 打开文件夹，删除 P:/EFI/ubuntu 文件夹
    6. 回到 diskpart，移除盘符 remove letter=p
  4. 删除 Ubuntu 所在分区
    1. 选择磁盘0，list partition 查看分区，一般未知分区为 Ubuntu 分区
    2. select partition 4 选择并通过 delete partition 删除分区
    3. 由于本机未为 Ubuntu 单独分配 EFI 分区，因此直接重复删除其他相关分区即可

安装 Ubuntu

  1. Win + R 运行 msinfo32 确认 BIOS 模式为 UEFI
  2. 下载 Ubuntu Desktop，此处版本为 22.04 LTS
  3. 通过 UltraISO 刻录至 U 盘
    1. 选择 文件 > 打开 打开 Ubuntu ISO 文件
    2. 选择 启动 > 写入硬盘映像，选择 U 盘 格式化 并 写入
  4. 安装 Ubuntu
    1. 开机时连续按 Del 键，进入 BIOS 系统
    2. 进入 Settings\Advanced\Windows OS Configuration 关闭 Secure Boot
    3. 选择开机启动项为 USB
    4. 选择 Try or Install Ubuntu 开始安装，按顺序安装
    5. 暂时选择不连接到互联网，不安装图形和Wi-Fi硬件驱动
    6. 选择 手动分区
      1. /Swap 分区与物理内存一致
      2. Ext4 挂载点 / 用于 Linux 系统文件及默认软件安装，分配后会自动选择 安装引导程序的设备
      3. Ext4 挂载点 /home 用于 Linux 用户文件
    7. 选择 安装 Ubuntu，等待安装完成
    8. 安装完成后，点击重启，提示移除安装媒介并按 Enter 键
    9. 重启后再次按 Del 进入 BIOS，设置开机启动项为 Hard Disk: Ubuntu
  5. 进入 Ubuntu 系统，连接网络，更新软件
